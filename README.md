# Rock Paper Scissors (Python)

A console-based Rock Paper Scissors game built in Python.

## Features
- User input validation
- Randomized computer choices
- Multiple rounds with scoring
- Dictionary-based outcome logic

## Concepts Practiced
- Control flow (loops, conditionals)
- Dictionaries
- Input validation
- Modular functions
- Basic program structure

This project was built as part of practicing core computer science fundamentals.
